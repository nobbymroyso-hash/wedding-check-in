# Wedding Guest Check-In App

## Components
- Google Sheet (Guests sheet)
- Google Apps Script backend (Code.gs)
- Mobile scanner frontend (index.html)

## Sheet Columns
Guest Name | Tracking Code | Accommodation | Pass URL | Status | Arrival Time | Check-In Time | Seated Time | Notes

## Deploy
1. Create Google Sheet and import CSV.
2. Open Extensions > Apps Script.
3. Paste Code.gs.
4. Deploy as Web App (Anyone with link).
5. Replace YOUR_APPS_SCRIPT_WEBAPP_URL in index.html.
6. Host index.html on any static host or Google Sites.
